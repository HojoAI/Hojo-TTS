# Hojo-TTS
Hojo-TTS is an open-source lightweight Text-To-Speech project by HojoAI team.
With only 0.08B parameters, that is, the parametere size is only 80M, Hojo-TTS can generate good enough quality speech.
Currently, Hojo-TTS supports both Chinese and English, and also support voice cloning with as little as a few seconds of audio.
