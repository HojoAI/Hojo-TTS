# Hojo-TTS-Light
Hojo-TTS is an open-source lightweight Text-To-Speech model by HojoAI team.
With only 0.08B parameters, that is, the parametere size is only 80M, Hojo-TTS can generate good enough quality speech.
Currently, Hojo-TTS supports both Chinese and English, and also supports voice cloning with as little as a few seconds of audio.
